 Frontend Implementation
 
 🚀 Project Overview
 A pixel-perfect, responsive web application built for a technical assessment. 
 This project demonstrates high-fidelity design translation using only vanilla HTML, CSS, and JavaScript, with no external frameworks or libraries.  
 
 🛠️ Key Features
 Image Gallery with Zoom: A custom vanilla JavaScript zoom lens that provides a high-resolution preview of product images on hover.  
 Sticky Header: A dynamic header that intelligently appears when scrolling beyond the first fold and disappears when scrolling back up.  
 Responsive Design: A fully fluid layout optimized for seamless performance across desktop, tablet, and mobile devices.  
 Interactive UI: Includes custom-built FAQ accordions, a manufacturing process navigation bar, and a product image carousel.  
 Modern Layouts: Built using semantic HTML5 and modern CSS practices, including Flexbox and CSS Grid.  
 
 📂 Project Structure
 index.html: Semantic structure for the landing page.  
 styles.css: Custom styling, layout logic, and responsive breakpoints.  
 script.js: Vanilla JavaScript for the sticky header, image zoom, and interactive components.
 
 🚦 Getting Started
 Clone this repository to your local machine.  
 Ensure the images/ folder contains the necessary assets exported from Figma.  
 Open index.html in any modern web browser to view the implementation.
